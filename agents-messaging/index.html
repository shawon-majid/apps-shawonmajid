First-time install notes

macOS: The app is not signed with an Apple Developer ID yet. The first time you open it, right-click the app in your Applications folder and choose Open → Open. After that, normal double-click works.

Windows: The installer is not yet code-signed. SmartScreen may show "Windows protected your PC" — click More info → Run anyway.

Need Intel Mac or older Windows builds?
  • The current build is Apple Silicon (M1/M2/M3/M4) only. Intel Mac support can be added — open an issue.
  • Windows build is x64. ARM64 Windows is not yet supported.
